==================================================================
Python EtherCAT Master lib
==================================================================

What is This. これはなに？
-------------------------------------------------------------------

これはEtherCATを試すためのサンプルソースです。

EtherCATのマスタープログラムをPython3(Socket)で記述しています。

pyMasterEtherCAT
